platziOkr b0.1

Real-Time OKR manager
https://platzi.firebaseapp.com/

Real-time HTML5 web app to administer group or personal Objective-Key-Results, built with Stylus and Angularfire (AngularJS + Firebase).

Requires:

* Node.JS
* Stylus
* Firebase
